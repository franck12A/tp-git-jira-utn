# ESCENARIO DEL PROYECTO

Escenario A – Análisis de Datos Climáticos.

El proyecto consiste en el procesamiento y análisis de datos climáticos históricos utilizando herramientas de control de versiones, gestión colaborativa y análisis de datos.

---

# OBJETIVO

Analizar registros históricos de temperatura global para obtener indicadores estadísticos básicos y generar visualizaciones que permitan interpretar la evolución climática a lo largo del tiempo.

---

# DESCRIPCIÓN DE LAS CARPETAS

## /datos
Contiene los datasets utilizados para el análisis climático.

## /scripts
Contiene los scripts desarrollados en Python para el procesamiento y análisis de datos.

## /resultados
Contiene gráficos, resultados y archivos generados por el análisis.

---

# FUNCIONALIDADES DEL PROYECTO

- Importación de datasets climáticos en formato CSV.
- Cálculo de temperatura promedio.
- Obtención de temperatura máxima y mínima.
- Generación de gráficos de evolución temporal.
- Exportación de resultados obtenidos.
